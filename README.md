## Описание

Приложение «Movie Library» — это простая GUI‑программа для управления личной коллекцией фильмов. Позволяет:
* добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга;
* просматривать все фильмы в виде таблицы;
* фильтровать фильмы по жанру и году выпуска;
* сохранять данные в формате JSON;
* загружать данные из JSON‑файла при запуске.

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.6+ и библиотека `tkinter` (входит в стандартную поставку Python).
2. Создайте папку проекта и поместите в неё файл `main.py`.
3. Создайте файлы `.gitignore` и `README.md`, как показано выше.
4. Откройте терминал в папке проекта.
5. Запустите приложение командой:
   ```bash
   python main.py
## Примеры использования
### Тест 1: Добавление фильма
### Заполните поля:

- Название: «Интерстеллар»

- Жанр: «Фантастика»

- Год выпуска: 2014

- Рейтинг: 8.6

- Нажмите «Добавить фильм».

- Проверьте, что фильм появился в таблице и появилось сообщение «Фильм успешно добавлен!».

### Тест 2: Фильтрация по жанру
- Добавьте несколько фильмов разных жанров.

- Выберите жанр в выпадающем списке (например, «Фантастика»).

- Нажмите «Фильтровать».

- Проверьте, что отображаются только фильмы выбранного жанра.

### Тест 3: Фильтрация по году
- Введите год (например, 2014) в поле «Фильтр по году».

- Нажмите «Фильтровать».

- Проверьте, что отображаются только фильмы за указанный год.

### Тест 4: Сброс фильтров
- Примените любой фильтр.

- Нажмите «Сбросить фильтры».

- Проверьте, что таблица снова показывает все фильмы.

### Структура проекта
- main.py — основной код приложения.

- movies.json — файл для хранения данных о фильмах (создаётся автоматически).

- .gitignore — настройки игнорирования файлов для Git.

- README.md — эта документация.

## Особенности
### Валидация ввода:

- год должен быть числом в диапазоне 1888–2030;

- рейтинг должен быть числом от 0 до 10.

- Автоматическое сохранение данных в JSON при добавлении фильма.

- Загрузка данных из movies.json при запуске приложения.

- Обработка ошибок при работе с файлами.
